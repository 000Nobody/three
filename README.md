# three
Dedicated to all things three - a projectinspired by [five.js](https://github.com/jackdclark/five)

## Install

```
git clone https://github.com/JasonTurley/three
python3
>>> from three import *
```

## Examples

**Addition**
```
three() + three() # 6
```

**Subtraction**
```
three() - three() # 0
```

**Multiplication**
```
three() * three() # 9
```

**Division**
```
three() / three() # 1
```

**Numbers**
```
three() # 3
squared() # 9
cubed() # 27
dozen() # 36
```
